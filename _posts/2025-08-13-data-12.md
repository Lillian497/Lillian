#  第一題 邏輯運算子 (Logic Operators)
```python
a,b,m=map(int,input().split())
##0,0  0,1  1,0  1,1
if a==0 and b==0:
    if m==0:
        print("AND")
        print("OR")
        print("XOR")
    else:
        print("IMPOSSIBLE")
elif a!=0 and b!=0:
    if m==1:
        print("AND")
        print("OR")
    else:
        print("XOR")
else:
    if m==0:
        print("AND")
    else:
        print("OR")
        print("XOR")
        

```
