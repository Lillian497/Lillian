# 第1題  三角形辨別 
```markdown
from sys import stdin
date=list(map(int,stdin.readline().split()))
date.sort()
a,b,c=date[0],date[1],date[2]
print(*date)
if a+b<=c:
    print("No")
elif a*a+b*b<c*c:
    print("Obtuse")
elif a*a+b*b==c*c:
    print("Right")
elif a*a+b*b>c*c:
    print("Acute")


```
